Welcome to the Noise tutorial

The data set we'll work on is TV shows. Every TV show is one document that contains the data about the show itself, like the name, length and air date. But also the cast and all the episodes.

Let's start simple and query for a show where we already know the exact name. Run the query with pushing the "Query"` button.

find {name: == "Seinfeld"} return .

A bit more advanced

This is the second step with some text

find {name: ~= "Star Trek: Voyager"} return .
